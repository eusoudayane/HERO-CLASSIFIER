# Classificador de Herói

Projeto do curso DIO para classificar heróis com base na XP.

## Como usar
1. Abra o arquivo `index.html` no navegador.
2. Digite o nome do herói.
3. Digite a XP do herói.
4. Clique no botão "Classificar Herói" e veja o resultado.

## Tecnologias usadas
- HTML
- CSS
- JavaScript
